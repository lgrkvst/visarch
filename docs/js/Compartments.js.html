Description

Compartments - a static closure for mapping between RSA compartment names and user-friendly compartment names. It also entails short descriptions as shown in (for instance) the autosuggest search window.

          name2RSA: function(name) {
			var match = false;
			var i = 0;
			while (!match && i < compartments.length) {
				if (name == compartments[i].name) match = compartments[i].compartment;
				i++;
			}
			return match;
		},
